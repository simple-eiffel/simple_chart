Overview

simple_chart provides a facade for loading CSV data and rendering it as ASCII visualizations. It supports horizontal bar charts with customizable width and character, and formatted ASCII tables with automatic column sizing.

Part of the simple_* ecosystem of focused, single-purpose Eiffel libraries with full Design by Contract support.

Quick Start

Installation

Set environment variable and add to your ECF:

-- Set SIMPLE_EIFFEL=D:\prod
<library name="simple_chart" location="$SIMPLE_EIFFEL/simple_chart/simple_chart.ecf"/>

Bar Chart Example

local
    chart: SIMPLE_CHART
do
    create chart.make
    chart.load_csv_string ("Name,Sales%NAlice,150%NBob,200%NCharlie,175")
    print (chart.render_bar_chart (1, 2))
end
Alice | ############################## 150
Bob | ######################################## 200
Charlie | ################################### 175

Table Example

local
    chart: SIMPLE_CHART
do
    create chart.make
    chart.load_csv_string ("Name,Age,City%NAlice,30,NYC%NBob,25,LA")
    print (chart.render_table)
end
+-------+-----+------+
| Name | Age | City |
+-------+-----+------+
| Alice | 30 | NYC |
| Bob | 25 | LA |
+-------+-----+------+

Features

CSV Data Loading

Load data from files or strings using simple_csv. Automatic header detection and column parsing.

Bar Chart Rendering

Horizontal ASCII bar charts with automatic scaling to maximum value. Customizable width and fill character.

Table Rendering

Formatted ASCII tables with borders, automatic column width calculation, and header separation.

Customizable Output

Set bar width (default 40), bar character (default #), and control rendering options.

Design by Contract

Full preconditions, postconditions, and class invariants for correctness verification.

Void Safety

Fully void-safe implementation with proper attached/detachable handling.

API Reference

SIMPLE_CHART (Facade)

Main entry point for all charting operations.

FeatureDescription
makeCreate chart instance with data loader and renderers
load_csv (path)Load CSV data from file at path
load_csv_string (content)Load CSV data from string content
render_bar_chart (label_col, value_col)Render bar chart using specified columns
render_tableRender data as formatted ASCII table
has_dataCheck if data has been loaded
last_errorGet last error message (if any)

BAR_CHART_RENDERER

Renders horizontal ASCII bar charts.

FeatureDescription
render (labels, values)Render bar chart from labels and values
set_bar_width (width)Set maximum bar width (default: 40)
set_bar_character (char)Set bar fill character (default: #)
as_stringGet rendered output as string

TABLE_RENDERER

Renders formatted ASCII tables with borders.

FeatureDescription
render (headers, rows)Render table from headers and data rows
as_stringGet rendered output as string

CSV_DATA_LOADER

Loads and provides access to CSV data.

FeatureDescription
load_from_file (path)Load CSV from file
load_from_string (content)Load CSV from string
row_countNumber of data rows (excluding header)
column_countNumber of columns
headersGet header row as list of strings
all_rowsGet all data rows
column_values (col)Get column values as strings
column_as_numbers (col)Get column values as REAL_64
