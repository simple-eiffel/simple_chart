## Relevant Standards

### SVG (Scalable Vector Graphics)
- W3C SVG 1.1 specification
- XML-based vector image format
- Universal browser/tool support

### Data Formats
- CSV input via simple_csv
- JSON data support (future)

## Industry Patterns

### Chart Library Patterns (from D3.js, Chart.js, ECharts)
- Declarative data binding
- Separation of data and presentation
- Responsive scaling
- Consistent axis handling

### Visualization Best Practices
- Clear axis labels
- Appropriate color palettes
- Accessible contrast ratios
- Legend placement conventions
